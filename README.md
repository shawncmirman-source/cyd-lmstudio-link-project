# cyd-lmstudio-link-project
Creates a dashboard using CYD (ESP32-2432S028) to show model loaded, status, session tokens, total tokens, and gpu temp

Included Components of this Repo

cyd_bridge.py - bridge application for PC
launch_bridge.bat - set to autorun and hide terminal running cyd_bridge.py

Firmware - includes platformio config, YMMV considering the wide variety of devices, Upon creation of the next branch I will update in greater detail the pinout information and display configuration
